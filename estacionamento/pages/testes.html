Testes de Integração

Esta página executa testes automatizados para validar a integração entre o frontend e o backend com PostgreSQL.

Os testes verificam:

  • Conexão com a API
  • Autenticação (login/logout)
  • CRUD de Veículos
  • CRUD de Mensalistas
  • CRUD de Isentos
  • CRUD de Serviços
  • CRUD de Preços
  • Geração de Relatórios
  • CRUD de Usuários
